# FE Project Setup

### Install pnpm

```sh
https://pnpm.io/installation
```

### Install dependencies

```sh
cd ecom-fe-230312
pnpm i
```

### Compile and Hot-Reload for Development

```sh
pnpm run dev
```

### Lint with [ESLint](https://eslint.org/)

```sh
pnpm run lint
```

### Format code with [Prettier](https://prettier.io/)

```sh
pnpm run prettier
```
